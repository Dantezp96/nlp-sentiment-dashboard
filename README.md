# NLP Sentiment Analysis Dashboard

Real-time sentiment analysis powered by RoBERTa, built with Gradio.

## Features
- Single text sentiment analysis with confidence scores
- Batch analysis via CSV upload or multi-line input
- Interactive dashboard with pre-loaded datasets
- Word clouds, distribution charts, and temporal trends

## Tech Stack
- **Model**: cardiffnlp/twitter-roberta-base-sentiment-latest (RoBERTa)
- **UI**: Gradio
- **ML**: HuggingFace Transformers, PyTorch
- **Visualization**: Matplotlib, WordCloud

## Run Locally
```bash
pip install -r requirements.txt
python app.py
```
